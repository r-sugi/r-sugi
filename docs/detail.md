## 大手インフラ系会員サイト開発
### 期間
  2022年11月 – 現在(14ヶ月~)

### 内容
イメージとしては東京電力TEPCOのような会員制ウェブサイト。
- サイト会員 数百万件、MAU 数十万人、DAU 数万人

### プロジェクト規模
  1チーム平均 4〜6人 x 2チーム(フロントエンド, バックエンド)でのスクラム開発

### 担当業務
フロントエンド、バックエンド

### 使用技術
Next.js, Typescript, Apollo, tailwind, storybook, vitest, playwright
NestJS, GraphQL

### 担当フェーズ
基本設計以降全て

### 開発規模
14人(エンジニア 10人、PO 1人、デザイナー 3人)

### その他
バックエンド
- 認証機能(Oauth2 Oracle)
- セキュリティ診断用の自動化ワークフロー構築
  - コンテナー脆弱性診断
  - API脆弱性診断
- テストコード
  - トランザクション時のテストコードの調査・実装

フロントエンド
- 新規機能開発
- エラーハンドリングのロジック修正
- セッションストレージの設計改善
- jest -> vitest + msw2へのバージョンアップ
- デプロイ時間短縮の調査

開発チームへ貢献
- モブプロ文化の醸成
- フロントエンドサブリーダー（コードレビュー）
